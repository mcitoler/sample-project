# Sample Project
Template for data projects.
```
sample-project/
├── data
│   ├── external
│   ├── interim
│   ├── processed
│   └── raw
├── notebooks
│   └── example.ipynb
├── references
├── reports
├── secrets
├── sample_project
│   ├── data
│   │   ├── __init__.py
│   │   └── make.py
│   ├── models
│   │   ├── __init__.py
│   │   ├── predict.py
│   │   └── train.py
│   ├── viz
│   ├── example.py
│   └── __init__.py
├── docker-compose.yml
├── .editorconfig
├── .env
├── .env.example
├── .gitignore
├── Makefile
├── pyproject.toml
└── README.md
```
Based off [Cookiecutter Data Science](https://drivendata.github.io/cookiecutter-data-science/) and [Docker for Data Science](https://dl.acm.org/citation.cfm?id=3152672).
